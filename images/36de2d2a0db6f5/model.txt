@startuml
title マックス・ウェーバーの4つの行為類型の関係性

package "行為類型" {
    class 目的合理的行為 {
        - 明確な目的と効率的手段
    }
    class 価値合理的行為 {
        - 内在的価値を重視
    }
    class 感情的行為 {
        - 感情や衝動による行為
    }
    class 伝統的行為 {
        - 慣習や文化に基づく行為
    }
}

目的合理的行為 --> 価値合理的行為 : 対立\n(目的vs価値)
感情的行為 --> 目的合理的行為 : 阻害\n(衝動が冷静な判断を妨げる)
伝統的行為 --> 感情的行為 : 感情を喚起
価値合理的行為 --> 伝統的行為 : 批判または強化
目的合理的行為 --> 感情的行為 : 制御\n(感情を抑える)
伝統的行為 --> 目的合理的行為 : 非効率とみなされることも
価値合理的行為 --> 感情的行為 : 感情的動機を含む場合も

note "現実の行為は複数の動機が混在" as N1
N1 ..> 目的合理的行為
N1 ..> 価値合理的行為
N1 ..> 感情的行為
N1 ..> 伝統的行為

@enduml
