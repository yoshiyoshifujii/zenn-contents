テストでpanic!をキャッチしたいときがあるよね。

そんなとき、どう書くのか、忘れちゃうので、メモ。

https://doc.rust-lang.org/std/panic/fn.catch_unwind.html

```rust
use std::panic;

let result = panic::catch_unwind(|| {
    println!("hello!");
});
assert!(result.is_ok());

let result = panic::catch_unwind(|| {
    panic!("oh no!");
});
assert!(result.is_err());
```

こいつは、よく使いそうですね。
